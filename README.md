# MACHINE-VISION-AND-PATTERN-RECOGNITION-I

# Computational Photography

A collection of projects exploring core concepts in computational photography, including filtering, convolution, frequency analysis, and multi-scale image blending.

---

## Projects

### OpenCV Messing Around
Basic image processing using OpenCV, including image manipulation and foundational filtering operations.

### Convolution
Implementation of convolution from scratch with experiments on different kernels and their visual impact.

### Fourier Transformation
Frequency domain analysis of images, exploring low and high frequency components and their effects.

### Multiresolution Blending
Multi-scale image blending and hybrid image creation demonstrating perception across viewing distances.

### Pinspeck Camera Analysis
Geometric analysis of pinhole imaging, including derivations for dual pinhole systems and image formation.

---

## Highlights

- Implemented core image processing operations from scratch  
- Built hybrid images combining frequency components  
- Explored spatial vs frequency domain representations  
- Derived pinhole camera geometry mathematically  

---

## Tech Stack

Python • NumPy • OpenCV • Matplotlib • LaTeX
